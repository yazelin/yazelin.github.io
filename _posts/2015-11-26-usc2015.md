## 課程簡介

這是 2015 年在實踐大學開設的「創意互動導覽行動平台人才培育學程」專業課程，聚焦於**導覽路線規劃功能之實作**，結合互動科技與行動平台開發。

### 課程特色

**目標：** 開發互動導覽 Android APP，整合 AR 擴增實境技術

**理念：**
- 從實際應用案例學習
- 以專案為導向的實作課程
- 團隊協作與成果展示

**技術棧：**
- **Unity 3D**：跨平台遊戲引擎
- **Vuforia SDK**：AR 擴增實境開發套件
- **NGUI**：Unity UI 系統
- **Android SDK**：Android APP 開發

## 課程大綱

### 第一週 (11/26) ♥ - 互動導覽應用與互動裝置

```
千里之行，始於足下
```

**學習內容：**

1. **互動導覽應用實例**
   - **互動科技在觀光休閒的創意應用**
     - 博物館互動導覽系統
     - 景點 AR 導覽應用
     - 文化遺產數位保存

   - **電子書的創意應用**
     - 互動式電子書
     - AR 增強型閱讀體驗
     - 教育出版的創新應用

   - **卓智飛船專案開發設計**
     - 專案背景與需求分析
     - 技術選型與架構設計
     - 開發流程與團隊協作

   - **卓智的互動應用案例**
     - AR 擴增實境應用
     - Kinect 體感互動
     - 3D 射擊遊戲

2. **互動裝置簡介**
   - **Kinect**：體感互動裝置
     - 深度感測技術
     - 骨架追蹤
     - 手勢辨識

   - **Leap Motion**：手勢控制裝置
     - 高精度手部追蹤
     - 手指動作辨識
     - VR/AR 整合應用

   - **嵌入式系統及各式感測器**
     - Arduino / Raspberry Pi
     - 溫度、光線、距離感測器
     - IoT 物聯網應用

3. **開發工具簡介**
   - Unity 3D 遊戲引擎
   - Android Studio
   - Vuforia AR SDK
   - GitHub 版本控制

## 課程特色

### 1. 專案導向學習

以**互動導覽 APP** 為核心專案：
```
需求分析 → 設計規劃 → 實作開發 → 測試優化 → 成果展示
```

### 2. 團隊協作

- 分組專案開發
- 定期進度報告
- 組間經驗交流
- 協作解決問題

### 3. 實機操作

不只是模擬器，而是**實際的 Android 設備**：
- 真實的使用者體驗
- 實機效能優化
- 實際場景測試

### 4. 令人興奮的學習曲線

課程使用 ♥ 符號表示**令人興奮的程度**：
- 第一週 ♥：認識工具
- 第二週 ♥♥：開始實作
- 第三週 ♥♥♥：看到成果
- 第四週 ♥♥♥♥♥：成果展示！
