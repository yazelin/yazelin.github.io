## 前言

在電子製造業，記憶卡等元件的插件作業需要精確的力道控制。**反向錯放**是常見的品質問題：記憶卡若 180 度反向插入，會因阻力過大而損壞主機板與記憶卡。

本文介紹我們在 **2017 自動化展**展示的記憶卡插件系統，利用 **KUKA 力覺感知技術**，不需外掛昂貴的力覺感測器，即可即時偵測反向錯放，避免元件損壞。

## 應用場景

### 挑戰

**傳統插件問題：**
- 記憶卡需插入主機板插槽，需要精確的力道控制
- 記憶卡可能 180 度反向錯放，強行插入會損壞元件
- 人工檢查費時且容易疏漏
- 傳統力覺感測器成本高達 30-80 萬元

**需求：**
- 即時判斷插入阻力是否異常
- 偵測到反向錯放立即停止
- 自動 NG 分揀，無需人工介入
- 低成本方案

---

## 解決方案

### 力覺感知插件流程

```
1. 機器手臂夾取記憶卡
   ↓
2. 移動至主機板插槽上方
   ↓
3. 啟動力覺感知監控
   ↓
4. 垂直下壓插入
   ↓
5. 【正常插入】
   - 扭力值在正常範圍內
   - 完成插入動作
   - 記錄為 OK
   ↓
6. 【異常偵測】（影片 53-58 秒）
   - 扭力值異常上升（反向錯放）
   - 立即停止下壓
   - 記錄為 NG
   - 取出記憶卡，放至 NG 區
```

### 技術原理

**不外掛 Sensor 的力覺感知：**
- 利用 KUKA 機器手臂的**關節扭力感測器**
- 即時監控各軸關節的扭力數據
- 比對正常插入與異常插入的扭力差異
- 超過閾值立即判定為異常

**與傳統方案比較：**

| 項目 | 外掛六軸力覺感測器 | KUKA 扭力感測方案 |
|------|------------------|------------------|
| **硬體成本** | NT$ 300,000 - 800,000 | NT$ 0（軟體開發） |
| **安裝時間** | 需拆裝手臂，2-4 小時 | 當日安裝、設定、測試 |
| **維護成本** | 感測器易損壞，維修費高 | 無額外維護 |
| **投產速度** | 需校正與調試 | **當日即可投產** |

## 系統規格

### 硬體配置

| 元件 | 規格 |
|------|------|
| **機器手臂** | KUKA KR6 R900 |
| **控制器** | KR C4 Controller |
| **力覺感測** | **KUKA 關節扭力感測（無外掛 sensor）** |
| **夾具** | 記憶卡專用夾爪 |

### 性能指標

- **Sensing Update Rate**：< 20ms
- **軸向感知**：XYZABC 軸向力覺感知
- **反應時間**：異常偵測後立即停止（< 50ms）
- **成功率**：反向錯放偵測率 100%
